# cricketers_stats

A simple and efficient **C language project** that manages cricketers’ performance data using **file handling** and **structures**.  
This program allows users to **add**, **update**, **display**, **search**, and **delete** player records through a menu-driven interface.

---

## Features

✅ **Add Player** – Add a new player’s details such as name, jersey number, matches, runs, wickets, average, strike rate, and economy.  
✅ **Update Player** – Update an existing player’s statistics using their jersey number.  
✅ **Display All Players** – View all players’ records in a tabular format.  
✅ **Search Player** – Find a specific player by entering their jersey number.  
✅ **Delete Player** – Delete a player’s record permanently using their jersey number.

---

## Technologies Used

- **Language:** C  
- **Concepts:** Structures, File Handling, Conditional Statements, Loops, Functions  
- **Compiler:** GCC or any standard C compiler
